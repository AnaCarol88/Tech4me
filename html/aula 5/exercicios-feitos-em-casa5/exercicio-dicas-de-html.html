Dicas de HTML


  Hyperlinks

 A tag <a> define um hiperlink

O atributo mais importante da tag <a> é o elemento "href", que indica o destino do link. E o texto do link, que ficará entre as tags, é a parte que ficará visível ao leitor.
Ao clicar no texto do link enviará o leitor ao endereço URL especificado. E por padrão, a página vinculada será exibida na janela atual do navegador. Assim, para mudar
isso, o atributo "target" especifica onde abrir o documento vinculado. Usando o "_blank" o documento abrirá em uma nova janela ou guia.

 Código fonte:

  <a href="" target="_blank"></a>  


  Listas ordenadas

 A tag <ol> define uma lista ordenada

Uma lista ordenada começa com a tag <ol> sendo um contêiner para os itens da lista, estes que deverão cada um começar com a tag <li>.
Desse modo, os itens da lista serão marcados com números.

 Código fonte:

  <ol>           
    <li></li>    
    <li></li>    
    <li></li>    
    <li></li>    
    <li></li>    
  </ol>          


  Tabelas

 A tag <table> define uma tabela

Uma tabela em HTML consiste em células de tabela dentro de linhas e colunas.
Ela começa com a tag <table> como contêiner, para o cabeçalho se usa a tag <thead>,cada linha da tabela começa com um <tr>, e para cada célula do cabeçalho a tag <th>,
assim, por padrão, o texto na tag <th> fica em negrito e centralizado. Já para o corpo da tabela, se usa a tag <tbody>, usando <tr> definir as linhas e <td> para cada célula.
Para fazer o rodapé da tabela, se usa a tag <tfoot>, usando <tr> definir a linha e <td> para cada célula.

 Código fonte:

  <table>          

    <thead>        
      <tr>         
        <th></th>  
      </tr>        
    </thead>       

    <tbody>        
      <tr>         
        <td></td>  
      </tr>        
    </tbody>       

    <tfoot>        
      <tr>         
        <td></td>  
      </tr>        
    </tfoot>       

  </table>
